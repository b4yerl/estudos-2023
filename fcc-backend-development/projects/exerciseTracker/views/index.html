Criar novo user



Cadastrar novo exercício para o usuário



  • /api/users << POST >> Gera um novo usuário e devolve um JSON com username e id, guarde o id.
  • /api/users << GET >> Retorna um array com todos os usuários cadastrados
  • /api/users/:id/exercises/ << POST >> Gera um novo exercício atrelado a um determinado usuário
  • /api/users/:id/logs << GET >> Retorna um objeto com dados do usuário e todos os seus exercícios
  • /api/users/:id/logs?[from][to][limit] << GET >> Devolve os mesmos logs de cima, mas com os filtros indicados
